General user guide to the application

This Python Application is able to run on every application that support the latest version of python. Please make sure you have python installed. To do this, go to the official website and download python and make sure you have a code editor on your PC.
If you are using Thonny, please make sure to import all needed libraries in there aswell.

Different modules and libraries

For this Python Application you will need to install PILLOW. In case you do not have PILLOW pre-installed, see the command line for your terminal in the README file.
All other libraries that need import are in a line of code on the top. If you are experiencing any problems, please contact 204369@stmichaelcollege.nl

Using the application

The application name is main_journal_app.py
You can run this on your PC.
When you run the code in your editor, there will appear GUI screen. This is where majority of the code will be run and displayed.
You will see a welcome/homescreen with 4 buttons and an image.

The buttons

All buttons have different functions associated with them. You can press each of them. Please do not be alarmed if another window pops up.
To go back to the userguide, press "Userguide".

Generate Prompt

The top-most button is to generate a prompt. Pressing this button will open a "TopLevel" Window named "Random prompt".
Here you can press "new prompt" to display a prompt.

Please note that pressing the button again will give another prompt. Pressing the button will not save the prompts previously displayed.

To display a new prompt, press the button again.
To exit this screen, click the "close" button, generally found in the top left corner.

this is not a button in the window, but rather in the window.

Add Prompt

The second button is to add a new prompt to the database (DB). pressing this button will open a "TopLevel" window named "Entry window", with a message box and a "Publish" button.

Publish

The publish button will 'push' whatever is written in the message box to the DB.

Please note this is irriversible unless you delete the DB from your laptop
Do not be alarmed when you do delete the database. Running the code again will automatically make a database with 21 starting "basic" prompts.

After publishing, clear the message box if you want to add another prompt or close the window the same way you would the "Random prompt" window.

Userguide

This button will open this webpage in your browser.

Close window

When you want to close the window, you can press this button.
Pressing this button will give a pop-up application. Please follow the instructions on screen.

Yes

Pressing "Yes" will close the window in 5 seconds. Please read the terminal in your code editor.

This action is irriversible once started. To use the application agian, run the code.

No
Pressing "No" will NOT close the window. In the terminal of your code editor you will see a message displayed.
